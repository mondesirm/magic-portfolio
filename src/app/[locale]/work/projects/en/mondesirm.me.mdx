Redesign and modernization of the portfolio of [Malik MONDESIR](https://mondesirm.me/en/about), Web Engineer and Webmaster, highlighting skills, projects and professional services.

## Description

[MONDESIRM](https://mondesirm.me) is a professional portfolio that I have redesigned. The website offers a modern user experience and was designed to make me visible to recruiters and clients. It highlights my skills, projects completed, and the services I offer as a Web Engineer and Webmaster. I also share my discoveries in the field of web development and web design.

## Objectifs

- **Branding**: Create a consistent visual identity that reflects my services.
- **Site web**: Design an intuitive website to showcase services and simplify online appointment booking.
- **Visibilité**: Strengthen the sole proprietorship's online presence on social media and recruitment platforms.

## Technologies

- **Canva**: Creation of graphic resources.
- **Vercel**: Deployment and hosting of the website.
- **Next.js**: React framework for building the web application.
- **Once UI**: React component library for a modern and responsive user interface.
- **Cal.eu**: Appointment booking system and availability management with email notifications.
